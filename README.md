## 📱 About

**Krishnagiri Farmer's Diary** is an offline-first mobile app for farmers to track daily income and expenses. It also shows **real-time vegetable prices** from Hosur market.

### 🎯 Built For
- Farmers in Krishnagiri, Hosur, Denkanikottai, Uthangarai
- Small-scale vegetable and dairy farmers
- Tamil, Telugu, and English speaking users

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📊 **Daily Income/Expense** | Track every rupee earned and spent |
| 📈 **Monthly Summary** | See total income, expenses, and profit |
| 🥬 **Live Market Prices** | Real vegetable prices from Hosur APMC market |
| 🌐 **3 Languages** | Tamil, Telugu, English - 100% translation |
| 📱 **Install as App** | PWA - install on mobile like a native app |
| 🔌 **Offline First** | All diary entries work without internet |
| 🎨 **Themes** | Green, Blue, Orange, Dark mode |
| 📅 **Date Filter** | Filter entries by month |
